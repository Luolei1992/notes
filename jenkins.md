 - 需要java环境8或者11（环境变量配置）
 - 命令行切换到jenkins文件目录下执行` java -jar jenkins.war`可能需要等很长时间下载插件
 - 成功后localhost:8080会有提初始密码位置，复制粘贴进入jenkins，设置用户，选择推荐插件
